Conceitos Básicos

O que é C? C é uma linguagem de programação que permite criar programas de computador, desde calculadoras simples até sistemas mais complexos.
Estrutura de um programa Todo programa em C começa em um ponto principal chamado main. É nele que o computador começa a executar o código..

Variáveis

    Variáveis são como caixinhas que guardam informações. Por exemplo:
Exemplo: Dizer Olá!
Números inteiros (int): 5, 10, -3 Números com vírgula (float): 3.14, 1.75 Letras (char): 'A', 'B'
    Mostrar e receber informações
    Para mostrar algo na tela, usamos printf.
    Para receber informações do usuário, usamos scanf.
    Exemplo: “Digite sua idade” e o computador lê o número que você digitar.

Operações, Decisões e Funções

    Podemos fazer contas: soma, subtração, multiplicação e divisão.
    Podemos tomar decisões usando if (se) e else (senão).
    Podemos repetir tarefas usando for ou while.
    E as Funções são pequenos blocos de código que realizam tarefas específicas e podem ser reutilizados várias vezes.
