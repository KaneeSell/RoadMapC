Operadores em C

Em C, operadores permitem manipular valores de variáveis para realizar cálculos, comparações e tomadas de decisão.

Operadores Aritméticos

+

Adição
int soma = 10 + 5;

-

Subtração
int sub = 10 - 5;

*

Multiplicação
int mult = 10 * 5;

/

Divisão
int div = 10 / 2;

%

Resto da divisão inteira
int resto = 10 % 3;

Operadores Relacionais

Usados para comparar valores, retornando verdadeiro (1) ou falso (0).

Exemplos

a == b // Igual a != b // Diferente a > b // Maior que a < b // Menor que a >= b // Maior ou igual a <= b // Menor ou igual

Operadores Lógicos

Combinam expressões lógicas:

&&

E lógico (AND)
(a > 0 && b > 0)

||

Ou lógico (OR)
(a > 0 || b > 0)

!

Negação (NOT)
!(a > 0)

Analogia Simples

    Pense nos operadores como ferramentas:
  1. O + funciona como uma calculadora de soma.
  2. O == funciona como uma balança, verificando se os dois lados são iguais.
  3. O && é como uma porta com duas chaves: só abre se ambas forem certas.
