Primeiro Programa

Para criar o primeiro programa em C, você precisa de um ambiente de desenvolvimento integrado (IDE), como o Dev-C++ ou o Code::Blocks, para escrever, compilar e executar o código. O programa "Olá, Mundo!" é um bom ponto de partida, incluindo a biblioteca stdio.h para saída de texto e a função main para iniciar o programa.

Passos para escrever e executar seu primeiro programa

    Escreva o código: Digite o seguinte código no arquivo:
Exemplo: Dizer Olá!
#include <stdio.h> int main() { printf("Oi Pessoal, Tudo bem?!"); return 0; }
    Compile o código: No Dev-C++, clique em "Executar" > "Compilar" ou pressione F9. Isso traduz o código em linguagem de máquina.
    Execute o programa: Após a compilação, clique em "Executar" > "Iniciar" ou pressione F10 para ver a saída do programa.
